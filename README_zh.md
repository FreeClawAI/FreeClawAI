# FreeClaw

**写代码，不花 Token。**

从 AI 对话中抓取代码，一键保存到本地项目。不花 API Token，不用复制粘贴。

> **无需编译、无需安装依赖、无需注册账号。下载即用。**

---

## 安装（30 秒）

1. 点击 **Code → Download ZIP** 下载
2. 解压到任意目录
3. Chrome 打开 `chrome://extensions/`，开启**开发者模式**
4. **加载已解压的扩展程序** → 选择解压文件夹
5. 打开 DeepSeek Chat 页面，右上角出现 📁 和 💾 按钮

> ✅ 不需要 `npm install` | ✅ 不需要 `pip install` | ✅ 不需要 Docker

---

## 可选：本地文件服务

如需将文件保存到本地项目目录：

```bash
node server.js
```

仅依赖 Node.js 内置模块，**无需 npm install**。服务运行在 `http://127.0.0.1:8080`。

## 使用方式

### 寄生模式

直接在 DeepSeek Chat 页面使用。寄生在 AI Chat 页面，**不消耗 Token**。

#### 获取源代码文件

在 AI Chat 中提问时，使用以下提示词让 AI 按 FreeClaw 格式返回代码：

> 用h2标题显示文件相对路径，不要加任何额外文字，代码块里放完整源代码。文件名需要相对路径。相对于工作目录的文件名。比如xxx.js，你应该告诉我js/services/xxx.js。你必须这样做，否则我去找其他ai了。

将此提示词保存为 FreeClaw 模板，每次要求 AI 返回代码时一键复用。

AI 会按如下格式回复：

```
## js/services/example.js
``javascript
// 完整源代码
``

``
## panel.css
``css
/* 完整样式 */
```

FreeClaw 会自动识别 `h2` 标题中的文件名和后面的代码块。

#### 保存文件到本地

**方式一：极速保存（推荐）**

点击右上角 💾 按钮，直接弹出保存对话框。勾选文件 → 确认保存 → 完成。全程不打开主面板。

**方式二：主面板保存**

点击右上角 📁 按钮打开主面板：
- 🔍 **提取**：从 AI 回复中提取所有代码文件
- 📄 **预览**：点击文件查看内容
- 💾 **保存**：勾选文件后点击保存按钮（或按 Ctrl+S）
- 📤 **发送**：选择本地文件 + 提示词，发送给 AI

#### 核心操作

```
AI 回复了代码 → 点击 💾 → 勾选文件 → 确认保存 → 文件出现在项目目录
AI 回复了代码 → 点击 📁 → 查看/编辑 → 批量保存
本地代码 → 点击 📁 → 选择文件 → 发送 → AI 审查或修改
```

## 功能

- 🔍 **一键抓取** AI 回复中的所有代码文件
- 💾 **极速保存** 不打开面板，直接保存
- 💾 **批量保存** 到本地项目目录，VSCode 直接打开
- 📤 **发送本地代码** 给 AI 审查或修改
- 🔄 **差异对比** 覆盖保存前查看代码变更
- ✨ **代码格式化** 使用 js-beautify
- 📋 **提示词模板** 收藏常用提示词，一键复用
- 📁 **多工作目录** 支持多个项目目录切换

## 常见问题

**需要 API Key 吗？**

不需要。FreeClaw 寄生在 AI Chat 页面，消耗的是 AI 平台的免费额度。

**数据安全吗？**

所有代码保存在你自己的电脑上。Node 服务器只监听本地 127.0.0.1。

**支持代码格式化吗？**

支持。右键文件 → 格式化代码。使用 js-beautify（MIT License）。

**为什么快捷保存按钮没反应？**

确保页面上有 AI 回复的代码块（带 h2 标题 + 代码块格式）。如果提示"没有可保存的文件"，说明当前页面没有提取到代码。
